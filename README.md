# 📦 Projeto React – Catálogo de Produtos

Este projeto é uma **aplicação web frontend desenvolvida com React.js e TypeScript**, criada com foco em **componentização**, **organização de código** e **boas práticas de desenvolvimento frontend**. A aplicação simula um **catálogo de produtos**, oferecendo fluxos distintos para **usuários comuns** e **administradores**.

O objetivo principal do projeto é consolidar conhecimentos em **React moderno**, estruturação de aplicações escaláveis e navegação entre páginas, servindo como base para futuras evoluções.

---

## 🎯 Objetivos do Projeto

* Praticar o uso de **React.js com TypeScript**
* Aplicar **boas práticas de componentização**
* Estruturar um projeto frontend escalável
* Trabalhar com **rotas e navegação**
* Simular fluxos diferentes de usuários (cliente e administrador)

## ▶️ Como Executar o Projeto

### Pré-requisitos

* Node.js instalado (versão LTS recomendada)
* Gerenciador de pacotes npm ou yarn

### Passos

```bash
# Clone o repositório
git clone https://github.com/Rafael540/Projeto-React.git

# Acesse a pasta do projeto
cd Projeto-React

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Após isso, a aplicação estará disponível em:
`http://localhost:5173`
